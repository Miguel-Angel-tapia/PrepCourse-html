Maquetacion: consiste en transformar el diseño de una web 

En HTML, existen muchas etiquetas que se utilizan para crear y organizar el contenido dentro del cuerpo de la página. Estas etiquetas de maquetado incluyen encabezados (
 a 
), párrafos (
), listas ordenadas (
    ) y listas no ordenadas (
      ), enlaces (), imágenes (), tablas (), formularios () y muchos más.


      : Define la cabecera de una sección o de todo el documento. Contiene elementos como el logotipo, el título de la página y la navegación principal.


      : Define una sección de navegación. Se utiliza para agrupar enlaces de navegación, menús o barras de navegación.


      : Define el contenido principal de una página web. Debe haber una sola instancia de esta etiqueta en un documento HTML.


      : Define un contenido independiente y autónomo en una página web. Puede ser un artículo, una publicación de blog, un comentario, etc.


      : Define contenido relacionado pero no esencial para la comprensión del contenido principal. Puede contener barras laterales, anuncios, enlaces relacionados, etc.


      : Define el pie de página de una sección o del documento. Contiene información como derechos de autor, enlaces a políticas, información de contacto, etc.


      : Define una sección temática dentro de un documento. Ayuda a estructurar el contenido de manera más significativa.
