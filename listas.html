Elemento 
  •  (Elemento de Lista):

Definición: define cada elemento de la lista, ya sea en una lista ordenada (
      ) o una lista no ordenada (
        ).
Ejemplo:
          
    
        • Elemento 1
          • 
    
        • Elemento 2
          • 
    
        • Elemento 3
          • 
        
        


Elemento 
           (Lista No Ordenada):

Definición: define lista no ordenada, donde los elementos se presentan sin un orden específico. Se representan típicamente con viñetas o marcadores (puntos).
Ejemplo: 
            
    
          • Elemento 1
            • 
    
          • Elemento 2
            • 
    
          • Elemento 3
            • 
        
          

Elemento 
             (Lista Ordenada):

Definición: define lista ordenada, donde los elementos se numeran secuencialmente o se etiquetan con letras u otros caracteres en un orden específico.
Ejemplo:
              
    
            1. Elemento 1
              2. 
    
            2. Elemento 2
              3. 
    
            3. Elemento 3
              4. 
       
            

Elemento 
             (Lista de Definición):

Definición: Este elemento se utiliza para crear una lista de definición, que consta de pares de términos (
            ) y definiciones (
            ).
Ejemplo:
            
    
            Término 1
            
    
            Definición 1
            
    
            Término 2
            
    
            Definición 2
            
      
            

Elemento 
             (Término de Lista de Definición):

Definición: Este elemento se utiliza para definir el término o concepto en una lista de definición (
            ).
Ejemplo: 
            
    
            Término 1
            
    
            Definición 1
            
    
            Término 2
            
    
            Definición 2
            
        
            

Elemento 
             (Definición de Lista de Definición):

Definición: Este elemento se utiliza para definir la descripción o definición de un término en una lista de definición (
            ).
Ejemplo:
            
    
            Término 1
            
    
            Definición 1
            
    
            Término 2
            
    
            Definición 2
            
       
            

casos de uso para saber en qué situaciones utilizar dichas listas:

Toda enumeración de pasos o secuencias a seguir en orden.

Cualquier enumeración de propiedades o características técnicas de una persona o producto. 

Las galerías de imágenes (al tener todas las fotos la misma información supongamos título, imagen y descripción y por formar todas parte del mismo álbum).

Las botoneras de todo sitio web (ya que cada botón forma parte del listado de opciones que tenemos para navegar nuestra página).
