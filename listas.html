             (Lista No Ordenada):

Definición: define lista no ordenada, donde los elementos se presentan sin un orden específico. Se representan típicamente con viñetas o marcadores (puntos).
Ejemplo: 
              
  * 
            • Elemento 1
              • 
  *
            • Elemento 2
              • 
  *
            • Elemento 3
              • 
        
            

Elemento 
               (Lista Ordenada):

Definición: define lista ordenada, donde los elementos se numeran secuencialmente o se etiquetan con letras u otros caracteres en un orden específico.
Ejemplo:
                
  1 
              1. Elemento 1
                2. 
  2
              2. Elemento 2
                3. 
  3
              3. Elemento 3
                4.

casos de uso para saber en qué situaciones utilizar dichas listas:

Toda enumeración de pasos o secuencias a seguir en orden.

Cualquier enumeración de propiedades o características técnicas de una persona o producto. 

Las galerías de imágenes (al tener todas las fotos la misma información supongamos título, imagen y descripción y por formar todas parte del mismo álbum).
