Etiquetas de Imagen:
Los formatos admitidos por esta etiqueta son: jpeg, gif, png, apng, svg, bmp, bmp ico, png ico.

: Se utiliza para mostrar imágenes en una página web. 
Atributos clave:
src : (ruta/fuente de la imagen).
alt : (texto alternativo describe la imagen,importante para la accesibilidad).
title: (Agrega título a la imagen, se muestra cuando el usuario pasa sobre ella).
